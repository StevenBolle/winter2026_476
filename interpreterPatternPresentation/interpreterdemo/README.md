# Interpreter Pattern Demo
This program is an implementation of the interpreter design pattern. Specifically, it is recursively checking the nodes of an
AST and building PEMDAS correct arithmetic from a formally defined language. Currently this recognizes and correctly processes:
-addition
-subtraction
-multiplication
-division
-powers
-parentheses
-variables
-variables with persisting value assignments, like x = 10
